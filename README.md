# The Library Management System in React.js

## Github Project Link (For others):
https://github.com/SumitChhaparwal/Book-Management-App-Reactjs-

## Features & Uses
-**About Application**: 
1.HomePage: have landing page, category section that display different categories, and popular book section which only display the popular books, not all. 
2.BrowseBook page: this have search bar where you can search any book which available on booklist by its title or author name, category section who display different book according to its category like fiction or non fiction, and at the end you can see list of books that filtered by its category. 
3.BookDetails page: This page open when you click any book in the booklist, where you can clearly see proper details of book.
4.AddBook page: this page is used to add new book to the booklist that process is handled by RTK. Add book page have form where you can enter new book details but must be vailid otherwise its show errors in UI and form comes with proper vailidation to secure application state and ensure booklist have correct books data.
